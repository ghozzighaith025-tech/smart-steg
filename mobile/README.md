# GridDNA AI — Technician Mobile App

**Status:** Week 5 deliverable (placeholder)

React Native + Expo application for field technicians.

## Planned Features

- Authentication (JWT)
- Assigned maintenance tasks
- Equipment QR scan → asset history
- Alert notifications (FCM)
- Maintenance reports with photos + GPS
- Offline sync

## Bootstrap (Week 5)

```bash
npx create-expo-app@latest griddna-mobile --template blank-typescript
```

Stack: Expo Router, React Query, SQLite (expo-sqlite), expo-camera, expo-location.
